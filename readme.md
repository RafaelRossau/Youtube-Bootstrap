YouTube Clone Project
Este projeto é uma interface responsiva inspirada no YouTube, desenvolvida para praticar a estruturação de layouts complexos utilizando Bootstrap 5 integrado com CSS puro.

🚀 Tecnologias Utilizadas
HTML5: Estruturação semântica das páginas de início e visualização de vídeo.

CSS3 (Puro): Estilização personalizada para componentes específicos como a sidebar, navbar e área de reprodução.

Bootstrap 5.3.8: Utilizado para o sistema de grid, utilitários de espaçamento e componentes rápidos como botões e barras de pesquisa.

Bootstrap Icons: Biblioteca de ícones utilizada para representar as funções da plataforma.

📋 Funcionalidades Implementadas
Página Principal (main-page.html):

Barra de navegação fixa com pesquisa e ações do usuário.

Barra lateral (Sidebar) rolável com categorias e inscrições.

Grid de vídeos responsivo utilizando o sistema de colunas do Bootstrap.

Página de Vídeo (video.html):

Layout focado no player de vídeo (representado por containers customizados).

Seção lateral para sugestões de "Próximos vídeos".

Estilização Customizada:

Tema escuro implementado via CSS.

Efeitos de hover e transições nos menus laterais.

📂 Estrutura de Arquivos
Plaintext

├── main-page.html      # Página inicial com o feed de vídeos
├── video.html          # Interface de exibição de vídeo individual
└── STYLEs/
    ├── main-page.css   # Estilos específicos da home e sidebar
    ├── navbar.css      # Estilos da barra de navegação superior
    └── video.css       # Layout da página de reprodução
🛠️ Como Executar o Projeto
Clone este repositório para sua máquina local.

Certifique-se de manter a estrutura de pastas para que os arquivos CSS sejam carregados corretamente via /STYLEs/.

Abra o arquivo main-page.html em qualquer navegador moderno.

Projeto desenvolvido para fins de estudo em desenvolvimento Front-End.
